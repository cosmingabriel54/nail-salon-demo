# Atelier LUNE — concept pentru salon de manichiură și pedichiură

## Fișiere
- `index.html` — landing page
- `servicii.html` — servicii și prețuri
- `portofoliu.html` — galerie filtrabilă
- `echipa.html` — prezentarea echipei
- `programare.html` — formular de programare
- `styles.css`, `script.js`

## Personalizare înainte de utilizare
- Înlocuiește numele Atelier LUNE cu numele clientului.
- Înlocuiește prețurile, serviciile și numele echipei.
- Conectează formularul la Formspree, Netlify Forms sau un backend propriu.
- Elimină `noindex,nofollow` numai după acordul clientului.
- Înlocuiește fotografiile echipei cu fotografii aprobate de client.
